# 📁 Dmitry — Funnel, Traffic & Geo Analytics

## 📌 Описание

Папка содержит модули для:

* анализа пользовательской воронки (event-based funnel)
* анализа трафика и источников
* анализа маржинальности
* географической аналитики (города / континенты)

Это **низкоуровневый аналитический слой**, где формируются ключевые агрегаты для дальнейших моделей и дашбордов.

---

## ⚙️ Основные компоненты

---

### 1. 🔻 Event-based Funnel Analysis

📄 `Event-based buyer funnel analysis.ipynb`

Что считается:

* последовательности событий пользователей
* переходы между событиями (`event_i → event_j`)
* частоты переходов
* структура пользовательских путей

📄 Выход:

* `sankey_links_anonymous.csv`
* `sankey_links_authorized.csv`

Содержат:

* source / target
* количество переходов

📄 Визуализация:

* `sankey_anonymous.html`
* `sankey_authorized.html`

---

### 2. 🌐 Анализ источников трафика

📄 `source_control.ipynb`

Что считается:

* распределение пользователей по источникам
* вклад источников в выручку
* влияние браузеров

📄 Выход:

* `margin_by_source_and_browser.csv`
  → маржа по (source × browser)

* `total_margin_by_source.csv`
  → суммарная маржа по источникам

---

### 3. 📊 Временная аналитика

📄 `sessions_weekly_stats.csv`

Содержит:

* количество сессий
* активность пользователей
* агрегаты по неделям

📄 `weekly_margin_analytics.csv`

Содержит:

* динамику маржи
* временные тренды

---

### 4. 🌍 Географическая аналитика

📄 `Maps analisys.ipynb`

Что считается:

* распределение пользователей по географии
* анализ маржинальности по регионам
* выявление прибыльных/убыточных зон

📄 Выход:

* `city_margin_distribution.csv`
  → маржа по городам

* `city_margin_pareto.csv`
  → Pareto-анализ городов (top contributing cities)

* `continent_margin_summary.csv`
  → агрегированная маржа по континентам

* `continent_return_rates.csv`
  → уровень возвратов по континентам

📄 Дополнительно:

* `margin_by_continents/`
* `margin_by_continents.zip`

→ детализированные данные по континентам
